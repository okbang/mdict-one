Work order : Deliverables

The deliverables are products that needs to be given to the customer. Right after the creation of any project, we must first define and estimate the size of the products before specifying that some of them are deliverables.

For more information about creating products please refer to Creating a product

Once that one or more products are created (and that one or more products are not defined as deliverable), and if you have the manage right on the Work order the Add new button will be available in the Deliverables page.

Adding a deliverable

Click on the button Add new on the bottom of the deliverable list, a new page will appear and let us input the following information :

- Deliverable : You can choose among the list of products that are not deliverable yet, the one that you want to define as deliverable.

- Delivery location: Since each deliverable must be sent to the customer, please input here the address needed.

- Note : Free text.

Click on Ok button to add the deliverable.

Deleting a deliverable

Deleting a deliverable can be done by clicking on the leftmost column of row you wish to remove in the deliverable page. Please note that the product information is NOT deleted, it is just removed from the deliverable list. If you want to remove a product, please refer to Deleting a product section.
