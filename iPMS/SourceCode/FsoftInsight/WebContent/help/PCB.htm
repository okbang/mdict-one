PCB Search/Create

The PCB (process capability baseline) report can be generated of consulted from this page.
A PCB report is based on projects that belong to the group or organization that has been selected prior to access this page. The PCB is based on projects that have been closed during a specific timeframe and that belong to a certain lifecycle.

Main elements defining a PCB :

Year: The year in which the period is included

Period: Part of the year, S meaning semester and Q meaning quarter

Type: Type of projects closed during the given timeframe (year and period) the type can be a lifecycle or a user manual selection of projects (Custom type)

Filtering reports:

Below Existing PCB reports line you will find 3 drop down lists that will enable you to filter existing PCB reports (if any) according to the year, period, and type of report.
When a value is selected in one of the drop down list, the list below will automatically be filtered for the new criteria.

Viewing report:

Simply click on a line in the list box below the filter drop down lists and then click on the button view.

Browsing a PCB report:

After pushing view or create button, you will see the PCB Information page. In the same time some new sub-menus will be available under PCB report in the left menu.
Each sub-menu will take you to one part of the PCB report:
PCB Information
PCB Metrics
PCB Comments
PCB Conclusion

Creating a PCB report:

If you have management rights on PCB report you will see on the bottom of the page the necessary controls for creating a PCB report.
The 3 drop down lists for defining the projects included in the report (see above), a text box will enable you to give a name to the report. And the check boxes will let you choose which metrics you want to be included in the report.
