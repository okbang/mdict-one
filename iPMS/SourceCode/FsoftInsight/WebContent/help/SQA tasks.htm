Tasks

This page is used in order to plan and track the group  activities.
The goal is to collect tasks that people must perform. Some entries are created automatically based on projects information or other systems. Some others can be created manually.
In order to view the detail of a task, please click on it's name.
Depending on the type of the task, you will then see the corresponding project page (automatic tasks) or the manual update page (manual tasks).
