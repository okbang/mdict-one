Milestone report

There is one milestone report for each stage completed (the milestone report will only be available if the actual end date of a given stage is set in the schedule). The available reports,  if any, can be selected in the drop down list on top of the page.
Each milestone report will show data related to the elapsed time of the stage.

At the end of each stage the project points should be filled in the corresponding milestone report in order to actualize the values in the project home page. For more details please see the point regulations.
