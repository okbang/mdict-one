PCB Contents

Browsing a PCB report:

After pushing view or create button in the PCB search/create page, you see the PCB Information page. In the same time some new sub-menus will be available under PCB report in the left menu.
Each sub-menu will take you to one part of the PCB report:
PCB Information
PCB Metrics
PCB Comments
PCB Conclusion

Updating the report:

If you have the management right on the PCB report, you will be able to update the textboxes displayed in the PCB. In order to validate your entries in a page please press Save button. If you browse another section of the PCB before saving, your entries will be lost.

PCB Information:

The various element defining the report are specified here. Given the management right, the user can Delete a PCB report (button on the bottom of the page). Clicking on this button will delete the current report and comments belonging to this report but not the data on which the report is based.

PCB Metrics:

Here are displayed the metrics calculated for the previous period and the chosen period. For reference, the norms applying to the Organization or Group is displayed as well.
The achievement column is calculated based on the current period data and the norm.
Clicking on a metric name (2nd column) will take you to PCB Detail metric page.

PCB Detail metric:

This page focuses on the metric selected in the PCB Metrics page some specific comments are also available.

PCB Comments:

These comment are more general than the ones of PCB Detail metric. For convenience, a link below the title will let you open a new window displaying the PCB Metrics page on which the comments are based.

PCB Conclusions:

This page displays the actions to be taken according to the data analysis.
The last problem and last suggestion review refer to the previous period PCB (previous semester or previous quarter).
