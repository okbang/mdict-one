Group home

After having selected the group you want to work on (if you have several groups to choose between), you will see the group home page.
On the left menu, you can access PCB, Indexes (Product, Process, Technology ...) and Issues

Points

On this page you can see 3 indicators:
1-Number of staff: Number of people currently registered as staff the group see dashboard tool)
2-Busy rate: This value is approximated with the formula: current month timesheet effort/(elapsed time since beginning of month*(nFullTimeStaff +0.5*nCollaboratorStaff )).
3-Number of running projects: Is the number of projects (belonging to the group) that have had status "in progress" during the month.
 

The group points are based on the Project points  (you can click on the project label to drill down to project values)  and on values input in the Point administration page.
