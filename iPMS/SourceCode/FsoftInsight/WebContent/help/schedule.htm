Schedule

The schedule page is used to track the actual completion of tasks against the planning.

There are schedule sections to be managed in project progress:

1. Project schedule

2. Deliverable schedule

3. Test and review schedule

4. The schedule for quality activities

5. Training schedule

6. Other schedules

Review and test activities

Review and test activities can only be defined based on an existing product, see Creating a product for more info.
Based on their size unit, modules will be subject to test or not :
-    If the language is Excel or Word, the module will be considered as a document and wont have any test date.
-    If the module language is not Excel or Word then the module will be considered testable and the plan/actual test dates will be displayed
