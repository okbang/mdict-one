Plans

The plans are used to forecast data about:

Staff
Finance
Infrastructure
Product
Process.

Each section displays the related metrics for the organization and the concerned groups.
Groups will plan for a metric depending on their type. Some metrics will be available for organization planning only, some other metrics will be available for production or support groups.
 

Each metric can be planned for each month and for the year. Currently, only the month planning is used for other sections (ex: see the planned values of indexes).

Values should be planned for each group and for each month first, by checking the checkbox Automatic calculation the values for the year and the organization values for the month will be refreshed  automatically upon each change.

Each plan can be saved for any period, but be careful not to overwrite already defined plan
