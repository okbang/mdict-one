Points administration

The points administration page lets you input manually the points calculated for each group as well as for organization. Please refer to the point regulations for the details of calculation.

First please input the year you want to view/edit then click on button view. The page will then be refreshed with the year's values.

Values can only be edited month per month, after changing the values for a month, please select this month in the drop down list on the bottom of the points table, then click update button.
