Work units

Work units can represent an organization like "FSOFT", a group for example "G5" or a project for example "Project silverlake".

Importing projects

To be able to use projects previously created with Dashboard in Insight, you must import these projects. In order to do so, please click on the button import projects in the work units page (management rights required).

Creating a project :

In order to create a project, we must first create a work unit (WU) of type project.
To this we must login with administrative rights and click on Admin tab. Then on the left menu click on the Work units icon.
The current list of WU is displayed, click on the the Add new button on the bottom of the list. On the new page displayed please input:

- the chosen code for the project in the field Work Unit Name, this name must be unique in the WU list.

- select type project among the radio buttons

- choose which group the project belongs to on the combo Parent work unit name, if the desired group is not in the list please create the group first (see Creating a group ).

- Click on Ok in order to proceed.
