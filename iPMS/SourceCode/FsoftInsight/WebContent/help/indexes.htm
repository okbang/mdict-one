Indexes

The indexes are metrics calculated for the projects belonging to a group or an organization. They can be accessed from the left menu of the group and organization home pages.

If accessed from the organization home, the indexes will display values for group, if from group, the indexes will display values for projects.

A click on a group will show values for the projects belonging to the group, a click on a project will display the summary metric page of the project.

Product indexes

Summary
    The summary page will display all the Product metrics in one page, the detailed information is available upon clicking on one metrics name. Metrics are calculated for "in progress" projects during the selected period.

In progress projects
    All number of project which have been "in progress" during the given month. Planned value comes from plans

Production size
    Sum of the products size in UCP (Use Case Points) of the above projects. Note: the size will only be calculated for products with an actual release date that occurs before the reporting date

Requirement stability
    Average of the projects' Requirement stability. Planned value comes from norms

Acceptance rate
    Average of the projects' Acceptance rate.  Planned value comes from norms

Requirement completeness
    Average of the projects' Requirement completeness.  Planned value comes from norms

Timeliness
    Average of the projects' Timeliness.  Planned value comes from norms

Delivery schedule deviation
    Average of the projects' Delivery schedule deviation Planned value comes from norms.

Defect rate
    Average of the projects' Defect rate.  Planned value comes from norms

Leakage
    Average of the projects' Leakage.  Planned value comes from norms

Customer satisfaction
    Average of the projects' Customer satisfaction.  Planned value comes from norms

Non conformity product rate
    Average of the projects' Non conformity product rate.  Planned value comes from plans
