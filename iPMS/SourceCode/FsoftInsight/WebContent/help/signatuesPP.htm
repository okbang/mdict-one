Signatures pages

The signatures pages (one in Work order section, one in Project Plan section), provide a convenient way to gather online the agreement of the project contractors/parties. It is not about real signatures, but their virtual equivalent i.e. the proof that the party agreed to the contract.

Prerequisite is that all parties have a account on the application, and the view right or manage right on the Work order or Project Plan section involved . If not the case please refer to Creating a user profile section.

First we need to define whose signature (or online agreement) are needed, for this we must login the application with manage rights on Work order section or Project Plan section (depending on which you need the signatures ) and go to sub-page Signatures of these sections.

The signatures can be gathered for several sub-pages of the Work order or Project Plan, this is why you will see several lists with the corresponding topic on top of each.

Choose the topic you need the signature for, on the bottom of the corresponding list you see 2 buttons : Delete and Add New.

Adding a signature

On the right hand of the Add New button, there is a combo box with the list of the application users. Select the one you need the signature for, and click on Add New button. The page will be refreshed and the user selected will appear in the list.

Deleting a signature

Select one or more users in the list by clicking in the check boxes and delete their signature by clicking on delete. This action will not change anything in the user account, it will just remove him from the signatures list (you can re-add the user anytime by Adding a signature).

Signing

In order to give their approval to the Work order or Project Plan, the users whose signature have been added to the signature list can open the corresponding signature page and click on the button Approve. The application will track the approval date and status.
