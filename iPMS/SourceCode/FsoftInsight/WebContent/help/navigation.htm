Navigation

You will find below some general guidelines about how to use Insight

Browser configuration

Insight has been successfully tested on Microsoft Internet Explorer 5 & 6.

1-Please open menu Tools>Internet options>General>Settings, check either "Every visit to the page" or "Automatically"

2-If the Insight server is on you Intranet please open menu Tools>Internet options>Connections>LAN and in the section "Exceptions" add an entry ";InsightServerName*" where InsightServerName is the name of the Insight server.

Account

In order to be able to use Insight you must first get your user name and password provided by the Insight administrator.
Then go to the login page, and input this information in the appropriate fields then click on login icon. The following screen is the:

Welcome page

The screen is divided into 3 parts as follow:

part 1
part 2 part 3

First please select in part 1 which Home page you want to access:

- Project home
- Group home
- Organization home
- Administration home

You can also view your access rights and change your password by clicking on your name

Then you will be prompted for the particular organization, group, or project (called work unit) you need to access.

The next screen will display :

- in part 2: a menu to acess subsections
- in part 3: summary information related to the work unit

General navigation

Most information is presented in lists, in order to view the details of a row you can click on the first column of the row. You will then be able to update or delete the item selected. In case you want to add a new item, there will be, each time, if it is possible and if you have manage rights on the work order, a button Add new on the bottom of each list.

Colour code

You will find some information displayed with different colors in Insight:

Blue color highlights good performance
Red color highlights bad performance

Different users, different rights

Depending on the role granted to you on a organization/group/project, you might have one of the following access rights on the information :
No access  or 
Read only or
Add / Update / delete rights
