Products and size

This page is the only place in the application where you can create products, products are all outputs that the project needs to manage and trace. Deliverables and Review and test activities are based on Products and therefore can't exist without them.

Creating a product

First, please log in the application on an account with manage rights on Products and size section, then we click on the project tab, select the project if needed and click in the Products icon on the left menu. On the bottom of the product list click on Add New button. On the new page,  please input the following fields : 

- Name: Name of the product for example : Proposal 01-jan-03

- Work product: select the category of the product, in our example : Proposal

- Estimated size unit: the size estimation is mandatory when creating a product, in this field, please select the unit for quantifying the product size in our example : Microsoft Word LOT (LOT=line of text).

- Estimated size: the estimated size in the unit above mentioned.

The following fields are optional and should be filled after the product has been built :

- Re-estimated size: if a new estimation has been made after the first one, and that the first one is already approved, please do not update Estimated size, but input in Re-estimated size, the unit is still Estimated size unit. 

- Actual size unit: depending on the Estimated size chosen, one or more choices will be offered.

- Actual size: the real size, should be input during product review and updated after release, measured in Actual size unit.

- Reuse: the reuse percentage indicate how much didn't need to be built for this product.
For example: If reuse is 70, it means the team only spent effort for creating 30 % of Test cases v1.0.

- Description: free text

After filling the necessary fields click on Ok button to validate entry.

Deleting a product

First, please log in the application, then we click on the project tab, select the project if needed and click in the Products icon on the left menu . Click on the leftmost column of the product you want to delete, then you should see the details page of the product, if you have the manage rights on Products and size, the Delete button will show on the bottom of the page.

Note : Pushing button Delete will also delete the Work Order deliverable based on this product if there is, as well as Review and test activities for the product. If you just need to delete the Work Order deliverable, please see Deleting a deliverable.
